## start
1. 安装依赖
终端中打开,在当前目录输入
```bash 
npm install
```
2. 然后，启动服务器
```bash
node app.js
```
在浏览器中打开 http://localhost:3000

## jquery

1. $()  选择器
2. html() 重写dom
3. on() 动态事件绑定
4. off() 移除事件
5. prop() 获取dom元素的属性
6. removeClass() 移除class
7. addClass() 增加class
8. hasClass() 判断是否有该class
9. append() 尾部追加内容
10. scrollTop() 获取滚动条的位置 或者 设置滚动条的位置
11. height() 获取dom元素的高度
12. empty() 清楚div内的所有内容
